# 형변환

## 기본 형변환

- 단일 인수를 다른 데이터 타입으로 변환하는 함수

```python
int() # 정수형으로 변환
float() # 실수형으로 변환
str() # 문자열으로 변환
chr() # 문자(한 글자)으로 변환
bool() # boolean으로 변환
```

## 자료형 형변환

- map() 함수를 사용해 자료형 안의 모든 데이터의 타입을 바꾸는 방법

```python
list = ["1", "2", "3"] # 문자열 리스트
# "1", "2", "3"

map = map(int, list) # map(함수, 리스트or튜플) 함수를 이용해 리스트를 정수형으로 변환
# <map object at 주소값>

list = list(map) # map을 list 형식으로 변환
# 1, 2, 3
```
