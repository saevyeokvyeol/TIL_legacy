# CSS 적용 방법

```jsx
$("선택자[, 선택자, 선택자...]").css("속성", "값[ 값 값...]")[.css("속성", "값").css("속성", "값")];
/*
	메소드 체인 기법으로 속성을 작성할 때는 카멜 표기법과 케밥 표기법 어느 쪽을 사용해도 적용됨
	값을 입력하지 않을 경우 css 속성값을 가져옴
*/

$("선택자[, 선택자, 선택자...]").css({속성: "값", 속성: "값", 속성: "값"...});
/*
	이 방법으로 속성을 작성할 때는
	속성을 따옴표로 묶으면 케밥 표기법을, 속성을 묶지 않으면 카멜 표기법을 사용해야 함
*/
```
